Options of GPIB interface

You can change the communication parameters of GPIB interface. For changing the secondary adress click on menu item Options -> GPIB secondary adress. Enable and select the secondary adress over the ring. Then apply the settings by pressing the OK button.



For changing "end of message" (EOS) click on menu item Options -> End of message and select a character or disable EOS. For changing timeout click on menu item Options -> Timeout and select a time.



Hint
Standard GPIB parameters are:
No secondary adress
EOS: LF (line feed)
Timeout: 1 second
