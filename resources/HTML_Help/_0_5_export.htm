Export data and create init- and sequence-files

In order to store the scanned and respectively edited informations of the present hardware you have to export the data. On the bottom of the main window are three check-boxes set. They show which files are going to be created.



To start the export process, select in the menu bar File -> Export physical.ini.



In the following panel you can select the devices, that are supposed to be registered into the file. All Rohde & Schwarz devices are enabled. Click on the entry to disable the device. Click on the Export button to save the enabled files onto hard disk. Type file name in file select popups and press save.

Note
To append information about the device's system-channels to the "physical.ini" enable the check-box Export System I/O Channels.
If the checkbox Export all Application-Channels is marked the respective entries in the "application.ini" are not left on some samples for logical names, but list all possible channels.
The check box Export ABus must be enabled if the switch manager or EGTSL is used.
The check box Enable auto coupling for TS-PMB should be enabled if the switch manager or EGTSL is used.

Hint
If the sample.seq is marked, the both init-files are needed, too.

Hint
Press CTRL and S to start the descripted process above.
