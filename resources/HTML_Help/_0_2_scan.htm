Scan for devices on your CompactTSVP

To setup your Rohde & Schwarz ComactTSVP informations regarding the physical layer of the connected system have to be written into physical.ini and application.ini files. Therefore Spectral TG-ITS is going to scan the hardware. To start the scanning process click on menu item Edit -> New Scan.

Please wait while the program searches for devices on different interfaces. If the program is finished, all available devices are shown in the panel .

Note
If you use National Instruments PXI devices (DAQ), you must first update NI Measurement and Automation Explorer MAX. To make sure that all GPIB devices are correctly recognized, you can check the GPIB settings over menu item (Options).

Hint
You can hit the Start-Scan-Process-Button or use the short cut F12 to start the scan process.
