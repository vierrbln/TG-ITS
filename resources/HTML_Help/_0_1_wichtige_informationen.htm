Important information about TG-ITS TemplateGenerator

To work with this tool, you need the following software packages:

  • National Instruments CVI runtime library 8.5
    (comes with the distribution kit)
  • Rohde & Schwarz GTSL 2.7 or higher
  • Device Drivers for TS-PSYS (comes with R&S GTSL 2.7)
  • National Instruments VISA Runtime 4.0 (comes with GTSL Install Disc)
  • IVI Compliance Package 2.1
  • National Instruments DAQ library
