Edit parameters of found devices

After the available devices are recognized, you can change the parameters of each device. All devices are registered into the menu bar. Select the device in the menubar (see picture below). After you selected a device, a panel opens itself in which you can change all parameters.



For all registered Rohde & Schwarz devices standard parameters are set. You should not change these values. For all unknown devices you change the values in the panel. After changing the values press Apply changes. If you click Quit without saving no changes will be saved.

Note
The bench name must be unambiguous. Permitted signs are letters, figures, underline and dot.

Hint
For fast access on the parameters make a double click on the desired device in the list view.
