Add devices manually

In order to simulate devices, which were not found during scanning process you have the possibility to add devices to the list manually. Click on menu item Edit -> Add device manually and choose the device's type. The bus-type can't be edited after creation.

After adding the edit-panel appears and the parameters have to be entered. This panel can also be reached by double-clicking the new devices in the list as hinted in "change parameters of found devices". All the parameters have to be entried by hand.

Furthermore there is an additional option in the edit-panel of manually added devices, to remove them from the list. By the way this allows to identify manually added entries. To delete all the manually added devices click on menu item Edit -> Add device manually -> Remove ALL added devices.

Hint
In the end NI Testand of course will not work properly with the manually added entries, if there is actually no accordant hardware available.
